% Armeemodellregeln
\newcommand{\warriorvassalsdef}{%
Der Bonus aufs Kampfergebnis den jeder Spieler aus einem Duell erhält (von verlorenen Lebenspunkten und Overkill) wird um 1 reduziert bis auf ein Minimum von 0.%
}
\newcommand{\ironraindef}{%
Schussattacken die vom Modell in der Flanke oder dem Rücken des Ziels durchgeführt werden erhalten +1 zum Treffen und +1 Rüstungsdurchschlag.%
}
\newcommand{\infernalarmourdef}{%
Plattenrüstung.
Der Träger erhält \textbf{\aegis{} (5+, gegen \flamingattacks{})}.%
}
\newcommand{\flintlockaxedef}{%
Zählt als Nahkampfwaffe mit \textbf{\twohanded} im Nahkampf.
Nahkampfwaffenattacken, die mit ihr durchgeführt werden, erhalten +1 Stärke.
Kann wie eine Nahkampfwaffe verzaubert werden, aber nur Nahkampfwaffenattacken sind von der Verzauberung betroffen, sofern nicht anders angegeben.
Das Modell darf aus dem dritten Glied schießen (zusätzlich zum ersten und zweiten).%
}
\newcommand{\blunderbussdef}{%
Erleidet nicht den negativen Treffermodifikator von einer Stehen und Schießen Angriffsreaktion.%
}
