\newcommand{\boilingoil}{Kochendes Öl}
\newcommand{\boilingoildef}{%
Wenn eine oder mehr gleichzeitige Attacken mit \boilingoil{} treffen, erhält die Zielenheit einen \boilingoil{} Marker nachdem die Attacken durchgeführt wurden.
Wenn eine Einheit mit einem oder mehreren \boilingoil{} Markern von Schussattacken mit \ironrain{} anvisiert werden, zählen die schießenden Modelle für \ironrain{} als im Rückenbereich des Zies befindlich (nicht aber für andere Regeln wie Flächenattacke).
Entferne alle Marker von einer Einheit wenn sie \newrule{von einer oder mehreren} Attacken mit \ironrain{} getroffen wird oder am Ende der Schussphase, je nachdem, was früher eintritt.%
}
