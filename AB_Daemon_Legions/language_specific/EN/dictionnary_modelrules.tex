\newcommand{\immortaldenizensdef}{%
Casting rolls made by models in a Daemon Legions army with one or two dice gain a +1 Casting Modifier.  In addition, in a game involving at least one Daemon Legions Army, each player adds +2 Veil Tokens to their Veil Token pool in the Magic Phase during Siphon the Veil when they are the Active Player.%
}

\newcommand{\aegisdef}{%
Unit profiles in this Army Book contain an additional Characteristic, which corresponds to the units' Aegis Saves, shortened \Aeg{}. This Characteristic is treated as if the unit has the Personal Protection Aegis (X+) written on its profile, where X is the \Aeg{} Characteristic value. Not having an \Aeg{} value does not prevent a unit from being the target of an Aegis modifier.%
}

\newcommand{\greaterdominiondef}{%
While within range of the model's \commandingpresence, the number of Health Point losses from failed Break Tests of friendly units with at least one \rnf{} model is reduced by X, where X is equal to the unit's number of Full Ranks. X is \textbf{always} at least 1 and \textbf{never} more than 3.
%
}

\newcommand{\darkfiredef}{%
Armour Save rolls of \result{1}, \result{2}, \result{3}, and \result{4} are \textbf{always} considered failed when saving wounds caused by \darkfire{}.%
}

\newcommand{\smother}{Smother}
\newcommand{\smotherdef}{%
If the attack is allocated towards a model that has the same Height as the attacker, it gains +1 to hit and +1 to wound.%
}
